# Tea Yield Predictors

This repository contains models and tools for predicting tea yields based on various parameters.

## Features
- Model training
- Data preprocessing
- Yield prediction

## Getting Started
To get started, clone the repository and install the required dependencies:

```bash
git clone https://github.com/albertrono/Tea-Yield-Predictors.git
cd Tea-Yield-Predictors
```
